# Negocio_BigData
## Optimización del presupuesto escolar
El instituto ha decidido implementar un sistema de Big Data para optimizar su gestión económica. El objetivo es analizar los gastos e ingresos del centro educativo en tiempo real, detectar patrones de consumo, anticipar necesidades presupuestarias y tomar decisiones más eficientes.

A través de la recopilación de datos financieros, compras, consumo energético, mantenimiento e inversiones, el sistema ayuda a mejorar la transparencia, reducir costes y asignar los recursos de forma más equitativa entre departamentos.
